My Project Highlights


TTC Pay Application: Android Platform



A group project which aimed at building an android application for TTC. I played a role of software developer in this project and used android studio as developing platform to create multiple application screens. The app was linked to a SQL database to manage the data .


Electronic Products API


Project related to API Management concept. An API application was built using C# on the visual studio. The main purpose of the API is to allow users to search for different electronic products along with their price. The Apigee platform was used to manage the API and was publshed to AWS cloud service.


Shopper's Point Application


Shoppers Point is a web based application which aimed at developing a platform for selling and purchasing of the used products.The seller could upload the pictures of a product and the buyer can view the product details on the application. IT project management principles were applied to manage the project in timely and cost-effective manner. I played the role of a project manager in this project which helped me gain deep understanding of success -factors of a project.
